## 📖 About

This repository is a **public log** of my learning journey through the **DSA Immersion 2026** program — a structured, placement-focused program designed to build strong problem-solving, logical thinking, and software engineering skills using **Java, Data Structures & Algorithms, and Pseudo Code Analysis**.

> Every folder added = a new topic explored. Every commit = something new learned.

---

## 🎯 Program Goals

- 🧠 Build strong problem-solving & logical thinking skills
- 📋 Understand real coding assessment patterns
- ☕ Learn DSA using Java from fundamentals to advanced
- 🔍 Master code execution & dry run techniques
- 🐛 Improve debugging skills
- 🎤 Prepare for placement interviews

---

## 🗺️ Learning Approach

```text
Problem Understanding
        ↓
Logic Building
        ↓
Pseudo Code
        ↓
Dry Run & Execution
        ↓
Java Implementation
        ↓
Optimization
        ↓
Interview Preparation
```

## 📚 DSA Using Java — Weekly Syllabus

### Week 1 — Java Fundamentals & OOP
- Java Execution Flow, JDK / JRE / JVM
- Variables & Data Types, Operators & Conditionals
- Class & Object, Constructors, Encapsulation

**Deliverables:** ✅ OOP Mini Project · ✅ Java Fundamentals Assessment

---

### Week 2 — Advanced OOP
- Inheritance, Method Overloading & Overriding
- Polymorphism, Abstraction, Interfaces
- Static & Final, SOLID Principles

**Deliverables:** ✅ OOP Coding Assignment · ✅ Technical Viva

---

### Week 3 — Arrays & Strings
- Arrays, ArrayList, String Manipulation, StringBuilder
- Frequency Count, Palindrome & Anagram Problems

**Deliverables:** ✅ Array & String Coding Sheet · ✅ Mock Assessment

---

### Week 4 — Searching & Sorting
- Linear Search, Binary Search
- Bubble, Selection, Insertion, Merge Sort
- Time & Space Complexity

**Deliverables:** ✅ Sorting Contest · ✅ Complexity Assignment

---

### Week 5 — Linked List, Stack & Queue
- Singly / Doubly / Circular Linked List, Reverse Linked List
- Stack Operations, Queue Operations, Interview Problems

**Deliverables:** ✅ Linked List Lab · ✅ Stack & Queue Problem Set

---

### Week 6 — Recursion, Hashing & Trees
- Recursion, Backtracking Basics
- HashMap, HashSet
- Binary Trees, Tree Traversals, BST

**Deliverables:** ✅ Tree Traversal Assignment · ✅ Hashing Practice Sheet

---

### Week 7 — Advanced DSA & Placement
- Greedy Algorithms, Dynamic Programming
- Sliding Window, Two Pointer Technique
- Elite 25 Coding Questions, Company-Specific Problems, Mock Coding Round

**Deliverables:** ✅ Grand Mock Test · ✅ Mock Technical Interview
